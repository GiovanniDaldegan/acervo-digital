# Acervo Digital
Protótipo de acervo digital para a preservação e difusão de patrimônio cultural, ligado ao projeto de extensão da UnB "CAL 3D", em parceria com Casa da Cultura da América Latina (CAL/DEX). As peças a serem expostas são gêmeos digitais, objetos 3D obtidos pelo escaneamento digital de obras do acervo da CAL.

## Estrutura do repositório
```
acervo-digital/
├── assets/                         # Recursos estáticos
│   ├── fonts/                      # Fontes
│   ├── img/                        # Imagens
│   └── models/                     # Modelos 3D (gêmeos digitais)
│                                   
├── src/                            # Páginas e códigos
│   ├── external/                   # Recursos externos
│   ├── pages/                      # Páginas html
│   │   └── acervo-digital.html     # Página principal
│   └── styles/                     # Estilos CSS
│       └── main.css                # Estilo geral das páginas
│                                   
├── LICENSE                         # Licença MIT
└── README.md                       # Este arquivo
```
